**A clean, modern, and real-time website monitoring tool** that helps you track the uptime of multiple websites with beautiful UI.

---

### ✨ Features

- ⚡ **Real-time Status Updates**
- ➕ **Add Any Custom Website**
- 🎨 **Modern & Attractive Dark UI**
- 📱 **Fully Responsive Design**
- 🔄 **Auto Refresh**
- 📊 **Live Status Cards**

---

### 🚀 Quick Start

```bash
# Clone the repo
git clone https://github.com/iamaryanbhalsing/Downdetector.git
cd Downdetector

# Install dependencies
pip install -r requirements.txt

# Run the app
python app.py
```
